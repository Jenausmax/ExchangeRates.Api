# BOT-0028: Статистика криптовалют + объединённая кнопка «Статистика»

## Цель

Добавить статистику криптовалют за период (до 6 месяцев). Кнопка «Статистика» показывает inline-меню с выбором: статистика валют или статистика монет.

## Поток взаимодействия

```
Кнопка «Статистика» (reply) → inline-меню:
  ├── «Статистика валют» → периоды: 3, 7, 14, 21, 30 дней
  └── «Статистика монет» → периоды: 3, 7, 14, 21, 30, 60, 90, 120, 150, 180 дней
```

### Inline-меню «Статистика»

| Кнопка | Callback | Действие |
|--------|----------|----------|
| Статистика валют | `stats_valute` | Показать inline-клавиатуру с периодами валют |
| Статистика монет | `stats_crypto` | Показать inline-клавиатуру с периодами крипто |

### Периоды «Статистика валют» (без изменений)

| Кнопка | Callback | Действие |
|--------|----------|----------|
| 3 дня | `period_3` | Статистика валют за 3 дня |
| 7 дней | `period_7` | Статистика валют за 7 дней |
| 14 дней | `period_14` | Статистика валют за 14 дней |

```

   Текущий: 8,456,789 ₽
```

## Изменения в KriptoService

| Параметр | Было | Стало |
|----------|------|-------|
| `KriptoConfig.HistoryRetentionDays` | 30 | 180 |
| `CryptoController` лимит hours | 720 (30 дней) | 4320 (180 дней) |

## Изменения в боте

### Новые интерфейсы

`IKriptoApiClient` — добавить метод `GetHistoryAsync(symbol, currency, hours, cancel)` для получения истории одной монеты.

### Затрагиваемые файлы

| Файл | Изменения |
|------|-----------|
| `KriptoConfig.cs` | `HistoryRetentionDays` 30 → 180 |
| `CryptoController.cs` | Лимит hours 720 → 4320 |
| `BotPhrases.cs` | Новые константы: BtnStatsValute, BtnStatsCrypto, StatsMenuHeader |
| `StatisticsHandler.cs` | Разделение на 2 пути, inline-меню, обработка `stats_valute`, `stats_crypto`, `crypto_period_*` |
| `CommandService.cs` | Маршрутизация новых callbacks `stats_*`, `crypto_period_*` |
| `IKriptoApiClient.cs` | Новый метод `GetHistoryAsync` |
| `KriptoApiClientService.cs` | Реализация `GetHistoryAsync` |

## Обратная совместимость

- Команда `/statistics` — теперь показывает inline-меню выбора (валюты/монеты)
- Кнопка «Статистика» — аналогично
- Callback'и `period_*` — продолжают работать (статистика валют)

Не требуются.
