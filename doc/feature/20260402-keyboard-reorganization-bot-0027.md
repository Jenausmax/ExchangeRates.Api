# BOT-0027: Реорганизация reply-клавиатуры бота

## Цель

Упростить reply-клавиатуру бота: 9 кнопок (3 ряда) → 5 кнопок (2 ряда). Объединить связанные функции под зонтичные кнопки «Курсы» и «Настройки» с inline-меню.

## Reply-клавиатура

### Было (3 ряда, 9 кнопок)

```
[ Курс сегодня ] [ За 7 дней  ] [ Статистика ]
[ Валюты       ] [ Подписка   ] [ Помощь     ]
[ Новости      ] [ Крипто     ] [ Монеты     ]
```

### Стало (2 ряда, 5 кнопок)

```
[ Курсы ] [ Новости ] [ Статистика ]
[ Настройки ] [ Помощь ]
```

## Inline-меню

### Кнопка «Курсы» → inline-меню

| Кнопка | Callback | Действие |
|--------|----------|----------|
| Курс валют | `rates_valute` | Показать курсы валют за 1 день (бывш. «Курс сегодня») |
| Курс монет | `rates_crypto` | Показать курсы криптовалют в RUB (бывш. «Крипто») |

### Кнопка «Настройки» → inline-меню

| Кнопка | Callback | Действие |
|--------|----------|----------|
| Настройки валют | `settings_currencies` | Открыть выбор валют (бывш. «Валюты») |
| Настройки монет | `settings_crypto_coins` | Открыть выбор криптомонет (бывш. «Монеты») |
| Подписка | `settings_subscribe` | Открыть меню подписок (бывш. «Подписка») |

## Переименования

| Было | Стало | Контекст |
|------|-------|----------|
| «Курс сегодня» | «Курс валют» | inline-кнопка в «Курсы» |
| «Крипто» | «Курс монет» | inline-кнопка в «Курсы» |
| «Валюты» | «Настройки валют» | inline-кнопка в «Настройки» |
| «Монеты» | «Настройки монет» | inline-кнопка в «Настройки» |

## Удалено

- Кнопка «За 7 дней» из reply-клавиатуры (команда `/valutesevendays` остаётся доступной)

## Обратная совместимость

- Все slash-команды продолжают работать: `/valuteoneday`, `/valutesevendays`, `/currencies`, `/cryptocoins`, `/subscribe`, `/crypto`, `/news`, `/statistics`, `/help`, `/start`
- Reply-кнопки «Новости», «Статистика», «Помощь» — без изменений
- Inline-callbacks для существующей функциональности (toggle_*, save_*, sub_*, crypto_*, period_* и т.д.) — без изменений

## Затрагиваемые файлы

| Файл | Изменения |

| `BotPhrases.cs` | Новые константы: BtnRates, BtnSettings, BtnRatesValute, BtnRatesCrypto, BtnSettingsCurrencies, BtnSettingsCryptoCoins, BtnSettingsSubscribe. Переименование BtnValuteOneDay. Удаление BtnValuteSevenDays, BtnCrypto, BtnCryptoCoins, BtnCurrencies, BtnSubscribe из reply-кнопок |
